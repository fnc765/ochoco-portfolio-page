# HTTP Request Shortcuts によるおはつい登録手順

Android アプリ「HTTP Request Shortcuts」を使い、X（Twitter）の共有メニューからおはついツイートを2タップでデータベースに登録する方法です。

## 必要なもの

- Android スマートフォン
- [HTTP Request Shortcuts](https://play.google.com/store/apps/details?id=ch.rmy.android.http_shortcuts)（無料・オープンソース）

## 設定手順

### 手順1: グローバル変数を作成する

共有で受け取ったテキスト（ツイートURL）を格納するための変数を作ります。

1. アプリを開く
2. 右上の **歯車アイコン** → **「変数」**
3. **＋ボタン** で変数を新規作成
   - 変数名: `tweet_url`
   - 種類: **静的（Static）**
4. 作成した変数の **詳細設定** を開く
5. **「共有ダイアログからの値の受け取りを許可」** を **ON** にする
6. 受け取る値の種類で **「テキスト」** を選択
7. 保存する

> **補足:** この変数に共有されたURLが一時的に格納されます。値は保存されず、実行ごとに共有テキストで上書きされます。

### 手順2: ショートカットを作成する

1. メイン画面に戻り **＋ボタン** → **「通常のショートカット」** を選択
2. 基本設定:

| 項目 | 設定値 |
|------|--------|
| 名前 | `おはつい収集` |
| メソッド | `POST` |
| URL | `https://ohatui-db.pages.dev/api/collect` |

### 手順3: リクエストボディに変数を設定する

1. **「リクエストボディ / パラメータ」** セクションを開く
2. 以下のように設定:

| 項目 | 設定値 |
|------|--------|
| ボディタイプ | **カスタムテキスト** |
| Content Type | `application/json` |

3. ボディ入力欄に以下を入力する:
   ```json
   {"tweet_url":"{tweet_url}"}
   ```
4. `{tweet_url}` の部分は、入力欄横の **`{ }` ボタン**をタップし、手順1で作成した変数 `tweet_url` を選択して挿入するのが確実です

### 手順4: Direct Share を有効にする（Android 11以降）

1. **「トリガーと実行の設定」** セクションを開く
2. **「Direct Shareターゲット」** を **ON** にする

これにより、共有シートに「おはつい収集」が直接表示されるようになります。

### 手順5: 保存する

設定を確認し、ショートカットを保存します。

## 毎日の使い方

1. X アプリでおはついツイートを開く
2. ツイートの **共有ボタン**（↗）をタップ
3. 共有先から **「おはつい収集」** を選択
4. 自動でAPIにPOSTされ、通知で結果が表示される

## APIの動作

- ツイートURLから自動的にツイート本文・画像・投稿日時を取得します（FixTweet API 使用）
- おはちょこ/こんちょこ/こんばんちょこ等の挨拶ツイートのみ保存されます
- 挨拶以外のツイートは自動でスキップされます
- 既に登録済みのツイートは上書き更新されます（重複エラーにはなりません）

## トラブルシューティング

### 「適切なショートカットが見つかりません」と表示される

手順1の **「共有ダイアログからの値の受け取りを許可」** が ON になっていることを確認してください。

### 登録に失敗する

- ネットワーク接続を確認
- URLが正しいか確認（末尾にスペース等が入っていないか）
- ショートカットの「レスポンス表示」を有効にしてエラーメッセージを確認

### 代替手段: admin画面からの手動登録

アプリがうまく動かない場合は、以下の方法でも登録できます:

1. X アプリでツイートの共有ボタン → **「リンクをコピー」**
2. ブラウザで `https://ohatui-db.pages.dev/admin/` を開く
3. URLを貼り付けて登録
