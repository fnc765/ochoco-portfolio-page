
# 【実装計画書】Bikyu83.com 風シンプルWebサイト制作

## 🎯 最終目標
https://bikyu83.com/ を参考にした、HTML+CSS+JSのシンプルなポートフォリオサイトを作成する

## 📚 学習目的
- 素のHTML/CSS/JSの基礎を理解する
- レスポンシブデザインの実装を学ぶ
- 外部ライブラリ(CDN)の使い方を学ぶ
- CSSアニメーションの実装を理解する

---

## 🗂️ プロジェクト構造

```
html+css+js/
├── index.html          # メインHTMLファイル
├── css/
│   └── style.css      # スタイルシート
├── js/
│   └── main.js        # JavaScript（パーティクル設定等）
└── images/
    ├── background.jpg  # 背景画像
    └── profile.jpg     # プロフィール画像
```

## 🛠️ 使用技術スタック

| カテゴリ | 技術 | 用途 |
|---------|------|------|
| **構造** | HTML5 | マークアップ |
| **スタイル** | CSS3 | デザイン・レイアウト |
| **動作** | Vanilla JavaScript | インタラクション |
| **エフェクト** | tsParticles (CDN) | パーティクルアニメーション |
| **画像** | JPG/PNG/WebP | 背景・プロフィール画像 |

### JavaScript
- DOM操作
- イベントリスナー
- 外部ライブラリの使用

## 🎨 デザイン参考要素（bikyu83.com）

### 配色
- 背景: ダーク系 + ブラー
